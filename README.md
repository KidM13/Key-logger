# Keylogger Mini Project (Educational)

## Overview
This project is a cybersecurity learning exercise created to demonstrate how keyboard and mouse input can be monitored in a controlled environment.

built strictly for educational purposes only.

## Project Structure

README.md  
Project documentation.

main.py  
Implements the keyboard listener (keylogger).  
It captures keyboard input and logs keystrokes for learning and analysis purposes.

listen.py  
Listens to mouse movement events.  
Used to demonstrate how mouse activity can be monitored.

control.py  
Controls and manages both keyboard and mouse interactions.  
Acts as the coordination layer between input listeners.

log.txt  
Stores captured keyboard input generated by the keylogger.

## requirement
u need a python library called pynput to deal with the input that come from mouse and keyboard

to install pynput

`pip install pynput`

## to run the project
in command line or terminal for both linux and windows
1, cd "project folder"

2 `python3 main.py`
